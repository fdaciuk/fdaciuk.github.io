Curso Javascript Ninja

O objetivo desse curso é ensinar Javacript DE VERDADE! Se você já está a muito tempo tentando estudar sozinho, mas não consegue compreender essa linda linguagem, esse curso é para VOCÊ! A ideia principal do curso é dar um Treinamento Ninja, para que o aluno saia com todo o conhecimento necessário para se virar sozinho com a linguagem, sem depender de bibliotecas ou frameworks.

Quais são os requisitos mínimos para fazer o curso?

  • Um navegador (preferencialmente Chrome ou Firefox);
  • Um editor de textos de sua preferência (recomendo Sublime Text);
  • Conhecimento básico do seu sistema operacional;
  • Conhecimento básico de HTML e CSS.
  • Lógica de programação - Isso você aprenderá durante o curso :D

O que será mostrado no curso?

  • Introdução ao Javascript (estrutura léxica, tipos, valores e variáveis);
  • Expressões e operadores
  • Declarações e estruturas
  • Objetos
  • Arrays
  • Funções
  • Javascript no client side (browser)
  • Eventos
  • Ajax
  • E no final, faremos um projeto para conclusão do curso!

Quanto tempo irá durar o curso?

O curso terá, em média, 30h de duração. O tempo não é exato, pois iremos passar por todos os pontos citados acima, então a duração do curso pode variar para um pouco mais, ou um pouco menos. O importante é o aluno realmente entender o conteúdo passado :D

Como o curso será disponibilizado?

Esse curso será gravado. Cada aula terá, em média, 1h de duração. Toda segunda-feira, à partir de 30/03, será lançada uma aula, explicando uma parte do conteúdo, e lançando um desafio ao final da aula, para que o aluno se sinta provocado a estudar para alcançar o nível Ninja!

Mas o curso não vai demorar muito tempo assim?

Sim! Mas como eu disse acima: a ideia principal é que o aluno aprenda DE VERDADE! É possível que, em algumas semanas, sejam lançadas mais de uma aula, mas o foco será o aprendizado real.

E eu terei suporte durante o curso? Como vou tirar dúvidas?

Com certeza! Enquanto estiver acontecendo o curso, será dado todo o suporte necessário através desse repositório no Github. Você poderá criar issues referentes à sua dúvida, que serão respondidas nesse local, para que nada fique subentendido :D

A cda semana, avançaremos em um exercício, e toda dúvida sobre ele será debatida nesse repositório. Então, se você ainda não tem, crie uma conta no Github, dê star no projeto e se prepare :D
